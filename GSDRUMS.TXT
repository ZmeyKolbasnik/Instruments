This -txt- describes the changes made to the standard GS Drum Preset Maps
in the soundbank file "GSDRUMS.SF2" 

The intent of this Soundbank to replace some common sounds from the original
AWE32-GS sounds, with better sounding samples.
 
To hear the changes you must load this sbk 
and play it under MIDI channel 10!

Since this file is a OVER 1meg it is intended for
AWE32 users with extra ram! (>512K)
